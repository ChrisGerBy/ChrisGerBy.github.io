How to begin with canvas

HTML


<canvas id="canvas" width="500" height="300"></canvas>

JavaScript


let canvas = document.getElementById("canvas");
let ctx = canvas.getContext('2d');
We can create rectangles, lines, circles, texts and other shapes.

Rectangles

ctx.fillRect(x, y, width, height);


ctx.fillStyle = "#f92672";
ctx.fillRect(100, 50, 100, 100);
							

ctx.rect(100, 50, 100, 100);
ctx.stroke();
Filled rectangles and empty rectangles (only border).

Lines


ctx.moveTo(60, 60); // start
ctx.lineTo(200, 100); // end
ctx.stroke();
Don't forget method stroke().

Circles


ctx.beginPath();
ctx.arc(100, 100, 80,      0,           2 * Math.PI);
    // x,    y,   radius,  startAngle,  endAngle
ctx.stroke();

How to begin with SVG
SVG uses HTML and XML.

Rectangles
  • Width of the rectangle
  • Height of the rectangle
  • Style of the rectangle: bg-color, border width, border color.

Lines
  • Coordinates X and Y
  • Style of the line: color, width.

More possibilities of SVG

Polygon, star, ellipse.
